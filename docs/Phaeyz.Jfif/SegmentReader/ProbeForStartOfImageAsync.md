# SegmentReader.ProbeForStartOfImageAsync method

Probes the stream to see if the next bytes on the stream are the start of a JFIF image. Some files may have a series of JFIF segments one after the other.

```csharp
public ValueTask<bool> ProbeForStartOfImageAsync(CancellationToken cancellationToken = default)
```

| parameter | description |

| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

A task yielding `true` if the stream has the start of a JFIF image; otherwise `false`.

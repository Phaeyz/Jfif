| parameter | description |

| segmentAttribute | If the method returns `true` this will receive the segment attribute. Otherwise it will receive `null`. |

## Return Value

Returns `true` if the segment class type is valid and the *segmentAttribute* received the attribute; `false` otherwise.

## Remarks

This function always returns `false` if the segment class type does not derive from [`Segment`](../Segment.md) or if the segment class type does not have a public default parameterless constructor.

# JfifMetadata.ReadAllFromStreamAsync method

Reads all JFIF metadatas from the stream.

```csharp
public static ValueTask<List<JfifMetadata>> ReadAllFromStreamAsync(MarshalStream stream, 
    SegmentDefinitions? segmentDefinitions = null, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| stream | The stream to read the JFIF metadata from. |
| segmentDefinitions | Optionally segment definitions which maps segment keys to segment class types. Specify `null` to use the default segment definitions. |
| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

Returns a list of [`JfifMetadata`](../JfifMetadata.md) objects based on what has been discovered in the stream.

## Remarks

There may be multiple sequences of sequences of JFIF metadata in the stream, where each sequence is a start-of-image segment, followed by a series of segments, ending with an end-of-image segment. It is possible that an end-of-image segment to immediately follow a start-of-image segment to begin new JFIF metadata.

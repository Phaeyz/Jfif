# JfifMetadata.ReadFromStreamAsync method

Read a JFIF metadata from the stream.

```csharp
public static ValueTask<JfifMetadata?> ReadFromStreamAsync(MarshalStream stream, 
    SegmentDefinitions? segmentDefinitions = null, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| stream | The stream to read the JFIF metadata from. |
| segmentDefinitions | Optionally segment definitions which maps segment keys to segment class types. Specify `null` to use the default segment definitions. |
| cancellationToken | A cancellation token which may be used to cancel the operation. |

## Return Value

Returns a [`JfifMetadata`](../JfifMetadata.md) if the stream contains valid JFIF metadata, otherwise `null` if a start-of-image segment could not be detected.

## Remarks

It is expected the first segment be start-of-image and last segment be end-of-image.

# SegmentDefinitions.Add method (1 of 2)

Adds a mappings for a marker and identifier to a segment class.

```csharp
public void Add(Type segmentClassType, bool overrideExistingMappings = false)
```

| parameter | description |
| --- | --- |
| segmentClassType | The type of segment class to add mappings for. |
| overrideExistingMappings | If `true` and there are matching markers and/or identifiers, they will be removed and replaced. If `false`, an exception will be thrown. The default value is `false`. |

## Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | The current object is read-only. |
| ArgumentException | A configuration issue occurred with the segment class, or a mapping for matching markers and/or identifiers already exists. |

## Remarks

The segment class must implement [`Segment`](../Segment.md) and be decorated with [`SegmentAttribute`](../SegmentAttribute.md).

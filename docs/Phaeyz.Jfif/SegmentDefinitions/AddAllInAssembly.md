# SegmentDefinitions.AddAllInAssembly method

Adds all eligible mappings for markers and identifiers to a segment classes in an assembly.

```csharp
public void AddAllInAssembly(Assembly assembly, bool overrideExistingMappings = false)
```

| parameter | description |
| --- | --- |
| assembly | The assembly containing the segment classes to add mappings for. |
| overrideExistingMappings | If `true` and there are matching markers and/or identifiers, they will be removed and replaced. If `false`, an exception will be thrown. The default value is `false`. |

## Exceptions

| exception | condition |
| --- | --- |
| InvalidOperationException | The current object is read-only. |
| ArgumentException | A mapping for matching markers and/or identifiers already exists. |

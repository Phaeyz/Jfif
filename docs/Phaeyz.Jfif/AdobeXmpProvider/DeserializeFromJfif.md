# AdobeXmpProvider.DeserializeFromJfif method

Deserializes XMP out of JFIF metadata.

```csharp
public static string? DeserializeFromJfif(JfifMetadata jfifMetadata, 
    bool throwOnInvalidSamples = true)
```

| parameter | description |
| --- | --- |
| jfifMetadata | The JFIF which may or may not contain Adobe XMP segments. |
| throwOnInvalidSamples | If `true`, when XMP segments are being deserialized an exception will be thrown if invalid data is encountered. If `false` the XMP data will be ignored and no exception thrown. |

## Return Value

Returns the XMP deserialized from the JFIF metadata.

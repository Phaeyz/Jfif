# ExifProvider.SerializeToJfif method

Serializes EXIF into the JFIF metadata.

```csharp
public static void SerializeToJfif(byte[]? exif, JfifMetadata jfifMetadata, 
    int? maxBytesPerSegment = null)
```

| parameter | description |
| --- | --- |
| exif | The EXIF to serialize into the JFIF metadata. If this is `null` or empty, all EXIF segments are removed. |
| jfifMetadata | The JFIF metadata to receive the Adobe XMP metadata. |
| maxBytesPerSegment | The maximum number of bytes to use per segment. If `null`, the `App1Exif.MaxExifBytes` is used. The default is `null` and this parameter is typically used for testing. |
